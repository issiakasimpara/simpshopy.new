Instructions de Suppression des Données

SimpShopy - Droit à l'oubli et suppression des données

Dernière mise à jour : 28 janvier 2025

1. Introduction

Conformément au Règlement Général sur la Protection des Données (RGPD) et aux lois maliennes, vous avez le droit de demander la suppression de vos données personnelles. Cette page explique comment procéder.

2. Données que nous stockons

SimpShopy peut stocker les informations suivantes :

  • Informations de compte : Nom, e-mail, numéro de téléphone
  • Données de boutique : Nom de la boutique, produits, commandes
  • Données de transaction : Historique des paiements via Moneroo
  • Données WhatsApp : Numéro de téléphone pour les notifications
  • Données d'utilisation : Logs de connexion, préférences

3. Comment supprimer vos données

3.1 Suppression automatique via l'interface

  1. Connectez-vous à votre compte SimpShopy
  2. Allez dans "Paramètres" → "Mon compte"
  3. Cliquez sur "Supprimer mon compte"
  4. Confirmez la suppression en tapant "SUPPRIMER"
  5. Attendez la confirmation par e-mail

3.2 Suppression manuelle par e-mail

Si vous ne pouvez pas accéder à votre compte, envoyez un e-mail à :

E-mail : privacy@simpshopy.com

Sujet : Demande de suppression des données

Contenu requis :

  • Votre nom complet
  • L'e-mail associé à votre compte
  • Le nom de votre boutique (si applicable)
  • Raison de la suppression (optionnel)

4. Ce qui se passe après votre demande

4.1 Délais de traitement

  • Demande automatique : Suppression immédiate
  • Demande par e-mail : Traitement sous 30 jours
  • Confirmation : E-mail de confirmation envoyé

4.2 Données supprimées

  • ✅ Informations de compte (nom, e-mail, téléphone)
  • ✅ Données de boutique (produits, commandes)
  • ✅ Préférences et paramètres
  • ✅ Données WhatsApp (si activées)
  • ✅ Logs de connexion

4.3 Données conservées (obligation légale)

  • ⚠️ Données de transaction (7 ans - obligation fiscale)
  • ⚠️ Logs de sécurité (1 an - protection contre la fraude)

5. Suppression partielle

Vous pouvez également demander la suppression partielle de vos données :

5.1 Supprimer uniquement les notifications WhatsApp

  • Allez dans "Paramètres" → "Notifications"
  • Cliquez sur "Désactiver WhatsApp"
  • Votre numéro sera supprimé de nos serveurs

5.2 Supprimer uniquement les données de boutique

  • Allez dans "Paramètres" → "Ma boutique"
  • Cliquez sur "Supprimer ma boutique"
  • Vos produits et commandes seront supprimés

⚠️ Important

La suppression est irréversible. Une fois vos données supprimées, elles ne peuvent pas être récupérées. Assurez-vous de sauvegarder toutes les informations importantes avant de procéder.

6. Vérification de la suppression

Après la suppression, vous pouvez vérifier que vos données ont bien été supprimées :

  • E-mail de confirmation envoyé automatiquement
  • Impossibilité de se connecter avec vos anciens identifiants
  • Boutique inaccessible si elle était publique
